## Jogabilidade
	A jogabilidade deste jogo é baseado em alternativas sobre situações na qual o professor se insere.
	Possui apenas uma resposta correta, caso você erre, a fase irá se repetir.
	Cada alternativa tem notas e você arrecada pontos ou perde que serão contados no final.
## História
	A História é simples, você é um professor universitário de geografia lidando
	com certas situações da escola que envolvem questões sociais e convivência, você orienta seus alunos
	a lidar com problemas relacionados aos conflitos de ideias.
## Comandos de Programação
	Os comandos que utilizei são bem simples, são comandos básicos do python, sendo eles:
	print, if, elif, else, input e while. Conta-se também as varíaveis, que é um dos comandos mais interessantes
	pois armazena códigos num simples nome. O if é um operador condicional, ou seja, ele só funciona com uma condição,
	Se x acontece, logo y deve acontecer. Ele é importante nesse jogo de alternativas, pois o que você escolhe,
	printa algo, caso não seja a condição do if, há o elif, que é uma condicional para a negação da condição do if,
	ou seja "se se não", e assim vai até o else, que eu coloquei como última questão incorreta, a correta fica fora
	do While, pois a questão não pode se repetir com a resposta correta.
